Pipelines

Multi-LLM workflows that chain personas together.

Discovery Pipeline

/vern discovery <prompt>

Your idea goes through 5 passes across multiple LLMs. Each pass builds on the last. The original prompt rides along the entire chain so nothing gets lost.

MightyVern
Analysis
→
Vernile
Refinement
→
YOLO Vern
Chaos Check
→
MightyVern
Consolidation
→
Architect
Breakdown

Configurable via /vern setup — swap any persona or LLM at any step.

VernHole

/vern hole <idea>

Summon 5–12 random personas from the full roster. Each gives their unique take. Chaos synthesizes into... something. You choose how deep to go.

5-6: focused council 7-9: getting chaotic 10-12: full VernHole

New Idea

/vern new-idea <name>

Creates a standardized discovery folder with input/ and output/. Drop your reference files in input/, then run /vern discovery {name} when ready.
